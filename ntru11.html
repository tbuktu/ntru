NTRU 1.1 Release

Version 1.1 of NTRU was released December 2, 2011.
Changes since 1.0:

  • NtruEncrypt keys can now be deterministically generated from a passphrase
  • Performance improvements:
    • Faster NtruSign key generation
    • NtruEncrypt key generation can now use two threads
    • Slightly faster encryption and decryption
